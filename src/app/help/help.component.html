Proben

Übersicht zur Zuweisung von Gennummern für Tagesnummern. Dafür die Tagesnummer in das Eingabefeld eingeben/einscannen und mit Enter oder Tab bestätigten. Daraufhin wird die Gennummer in das Clipboard kopiert. Zeitgleich erscheint die Tagesnummer mit Gennummer, Ersteller und Datum unten als "Karte".

Zusätzlich können die Optionen Sofort Archivieren und Sofort Drucken ausgewählt werden. Bei Sofort Archivieren wird nach erfolgter Eingabe sofort ein Rackplatz zugewiesen. Das Feld Sofort drucken ist standardmäßig angewählt und sorgt dafür, dass nach erfolgter Eingabe, ein kleines und ein großes Etiketten gedruckt werden. Diese lassen sich ebenfalls durch einen Klick die Karte drucken.

Außerdem unterstützt die Software den Export der Proben von einem auswählbarem Datum. Dafür muss das Datum in das vorgesehene Eingabefeld eingetragen bzw. ausgewählt werden. Es wird eine .csv Datei zum Download angeboten die genutzt werden kann um die Daten nach MOLIS zuz exportieren.

Archiv

Hier können Proben durch Scan der Tagesnummer einen Archivplatz bekommen. Die Probe muss bereits eine valide Gennummer bekommen haben. Daraufhin kann die Probe in das Eingabefeld gescannt werden worauf ein Rackplatz vergeben wird.
