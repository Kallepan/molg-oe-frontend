Proben

Übersicht zur Zuweisung von Gennummern für Tagesnummern: Geben Sie die Tagesnummer in das Eingabefeld ein oder scannen Sie sie ein und bestätigen Sie mit Enter oder Tab. Die entsprechende Gennummer wird daraufhin in die Zwischenablage kopiert. Gleichzeitig wird die Tagesnummer zusammen mit der Gennummer, dem Ersteller und dem Datum unten als 'Karte' angezeigt.
Zusätzlich stehen Ihnen die Optionen 'Sofort Archivieren' und 'Sofort Drucken' zur Verfügung. Bei 'Sofort Archivieren' wird nach der Eingabe automatisch ein Rackplatz zugewiesen. Das Feld 'Sofort Drucken' ist standardmäßig aktiviert und bewirkt, dass nach der Eingabe sowohl ein kleines als auch ein großes Etikett gedruckt werden. Diese können auch direkt über einen Klick auf die Karte gedruckt werden.
Die Software ermöglicht auch den Export von Proben, die noch nicht exportiert wurden. Klicken Sie einfach auf den Export-Button, um alle nicht exportierten Proben zu exportieren. Nach dem Export werden diese Proben automatisch als 'exportiert' markiert. Sie erhalten dann eine .csv-Datei zum Download, die zur Übertragung der Daten nach MOLIS verwendet werden kann.

Archiv

In diesem Bereich können Proben durch Scannen der Tagesnummer einem Archivplatz zugewiesen werden. Beachten Sie bitte, dass die Probe bereits eine gültige Gennummer erhalten haben muss, bevor sie archiviert werden kann. Sobald die Tagesnummer der Probe in das Eingabefeld gescannt wird, wird automatisch ein passender Archivplatz zugewiesen.

Löschen

Zusätzlich verfügt die Software über einen separaten Reiter für das Löschen von Proben. Der Löschvorgang erfolgt ähnlich wie das Einscannen einer Probe. Navigieren Sie zum Löschreiter und geben Sie die entsprechende Tagesnummer der zu löschenden Probe in das Eingabefeld ein oder scannen Sie sie ein. Bestätigen Sie anschließend die Eingabe mit Enter oder Tab.
Nach Bestätigung wird die entsprechende Probe aus dem System gelöscht. Die gelöschten Proben werden automatisch aus der Datenbank entfernt und können nicht wiederhergestellt werden.
Bitte beachten Sie, dass das Löschen von Proben eine irreversible Aktion ist. Stellen Sie sicher, dass Sie die korrekte Tagesnummer eingeben, um unerwünschten Datenverlust zu vermeiden.!
